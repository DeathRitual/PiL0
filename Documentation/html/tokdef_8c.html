Detailed Description

Library for Token-Stream.

Contains data structures and functions to implement a linear token stream. Error handling ensures that on wrong behaviour a message will be print what went wrong. With the functions in this file new Tokens can be created and connected to each other, so they represent an FIFO (new elements will be added at the end of the stream, whereas the oldest element in the stream will be deketed from the top of the stream). Also there are functions to read the newest and oldest element of the stream.
